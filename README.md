# Snake Clash

브라우저에서 즐기는 slither.io 스타일 아레나 스네이크 게임. 의존성 없는 단일 HTML 파일(Canvas + 순수 JavaScript)이라 PC·안드로이드 태블릿의 크롬에서 URL만으로 바로 실행됩니다.

## 플레이 방법
- **조준**: 마우스 이동 / 터치
- **부스트**: 클릭 · 스페이스 (길이를 소모해 가속)
- **목표**: 제한시간 2분 안에 길이 250 도달 시 즉시 승리. 시간이 끝났을 때 1등이어도 승리.

## 규칙
- 상대의 머리를 내 몸으로 막으면 처치(질량 흡수)
- 내 머리가 상대 몸에 닿으면 사망
- 정면 충돌은 더 긴 뱀이 승리(비슷하면 둘 다 사망)
- 맵 경계 밖으로 나가면 사망

## 로컬 실행
```
python -m http.server 8123
```
또는 `index.html`을 브라우저로 직접 열기.

## 배포
GitHub Pages로 호스팅됩니다. 저장소 Settings → Pages에서 배포 후 공개 URL로 어디서든 접속 가능합니다.
